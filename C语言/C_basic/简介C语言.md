# 第一章：C语言简介
## 1：高级语言的概念：
- 机器语言：二进制表示，面向机器直接运行
- 汇编语言：符号化的机器语言，翻译后再运行
- 高级语言：类自然语言，面向问题翻译后再运行
## 2：编程语言的分类：
### 抽象等级：
- 机器语言
- 汇编语言
- 高级语言
### 程序设计方法：
- 面向过程：C
- 面向对象：Java,C++,Python,C#,Visual Basic.NET,PHP,JavaScript,R等
### 程序执行方式：
- 编译型语言：C,C++,高级语言通过编译器直接转化为机器语言，生成可执行文件（.exe）
- 解释型语言：Python,Java,PHP,JavaScript,SQL,R
- 混合型语言：Java,C#,Visual Basic
- 由汇编语言转化为机器语言的过程叫做汇编
- 由高级语言转化为汇编语言的过程叫做编译
- 边翻译边执行的过程叫做解析
### 运行时结构是否会发生变化：
- 动态语言：Python,PHP,JavaScript,R
- 静态语言：Java,C,C++,C#,Visual Basic.NET,SQL
### 变量是否需要指明类型：
- 强类型语言：Java,C,C++,C#,Visual Basic.NET
- 弱类型语言：Python,PHP,JavaScript,SQL,R

## 3：分代：
### 第一代：机器语言：数字化
### 第二代：汇编语言：符号化
### 第三代：高级语言：结构化、对象化
- 主要经历分为：
    - 非结构化程序设计
    - 结构化程序设计
    - 面向对象的程序设计
    - 可视化开发环境
    - 模拟仿真环境
      - 面向过程：C,BASIC,pascal
      - 面向对象：C++,java,C#
      - 面向对象：SQL
#### 第四代：数据库编程语言，如各种数据库的SQL语言：强智化
#### 第五代：人工智能语言，或者称之为类自然语言、或称为知识库语言：弱智化
#### 第六代：暂无定论
## 4：学习要领：
- **语法**：语言基本元素、基本构词方法、句法、结构等规则的集合
- **算法**：解决特定问题的一般方法
- **方法**：结构化程序设计的思想方法

## 5：C语言的发明
### 创始人：Dennis M Ritchie 和 Brain W Kernighan
### 背景：在B语言的基础上提出的
- B语言的作者：Ken Thompson
- C语言是由B语言的第二个单词的第一个字母来编写的
## 6：C语言的特点：
- C语言具有完善的功能和非常高的效率
- 可对硬件进行操作（介于高级语言和汇编语言之间的一种中级语言）但是属于高级语言
- 是结构化的程序设计语言
- 具有丰富的数据类型和运算类型
- 具有良好的移植型
- C语言书写灵活，表达简洁
### 优势：
- 适合编写操作系统软件
- 适合编写对运行效率比较高的系统
    - 设备的驱动程序
    - 高性能、实时中间件
    - 嵌入式领域
    - 并发程序设计
### 程序的基本组成部分：
- 程序是由函数组成的，函数是由语句组成的，主函数唯一，主函数名称为main,子函数可有可无，也可以有多个
- C语言的保留关键字必须为小写，用户构词可以是大小写混合使用
- C语言中语句采用;(分号)来进行终止
- 名词之间用一个或者几个space来进行分隔
## 7：C语言的词法
### C语言的字符集：
- 字母：52个字母，A,a等
- 数字符号：0-9
- 特殊符号：+-*/等
### 保留字
- 1: auto:
    - 自动存储类别，在函数内声明的局部变量，其存储期限为函数执行期间，作用域为函数内
    - 编译器会自动为其分配存储空间，并在函数返回时释放
    -     auto int x = 10
- 2: break
    - 终止循环语句或者switch语句，并跳转到循环或者switch语句的后面的语句
    -     while (1) {
          if (condition) {
            break;
          //其它语句
            }
          }
- 3: case:
    - 在switch语句中，用于匹配特定的常量值
    - 可以跟整形常量表达式、枚举类型的值、也可以跟字符串常量表达式
- 4：char
    - 声明字符型常量
    - 示例：
    -     char ch = 'A';
- 5：const
    - 声明只读变量，其值在程序运行期间不可更改
    - 示例
    -     const int MAX_SIZE = 100;
- 6：continue
    - 跳过循环剩余的语句，并且开始下一次的循环
    - 示例：
      -     for(int i = 0; i < 10; i++){
            if (i % 2 == 0){continue;//其他代码；}}
- 7：default
  - 在switch 语句中，用于匹配不符合任何case的情况
  - 示例：
    -     switch(x){
          case 1:
          //执行代码;break;
          case 2:
          //执行代码;break;
          default:
          //执行代码;
          }
- 8：do
  - 开始do-while循环
  - 示例：
    -     do {
          //循环体；
          }while(condition);
- 9：double
  - 声明双精度浮点型变量
  - 示例：
    -     double pi = 3.1415926;
- 10：else
  - 在if-else 语句中，用于指定当条件不成立时执行的语句
  - 示例：
  -     if (condition){
  -     //执行代码;
  -     }else{
  -     //执行代码;}
- 11：enum:
  - 声明枚举类型，用于定义一组命名的整形常量
  - 示例
  -     enum Color{RED,GREEN,BLUE};
- 12：extern
  - 声明在其他文件中定义的变量或者函数
  - 示例：
  -     extern int global_var;
  -     extern void func();
- 13：float
  - 声明单精度浮点数
  - 示例
  -     float pi = 3.14159f;
- 14：for
  - 开始for循环
  - 示例：
  -     for( int i = 0; i < 10; i ++){
  -     //循环体}
- 15：goto
  - 跳转到指定标签处执行代码
  - 示例:
  -     goto label;
  -     //其他代码;
  -     label:
  -     执行代码;
- 16：if
  - 开始if - else 语句
  - 示例
  -     if (condition){
  -     //执行代码;
  -    }else{
  -    //执行代码;}
- 17：int
  - 声明整形变量
  - 示例：
  -     int x = 10;
- 18：long
  - 声明长整形变量
  - 示例：
  -     long long_num = 10000000000000000000;
- 19：register:
  - 声明寄存器变量，用于将变量存储在寄存器中以加快访问速度
  - 示例：
    - register int x = 10;
- 20：return
  - 从函数返回值
  - 示例： 
  -    int add(int a, int b){
  -    return a + b;}
- 21：short
  - 声明短整型变量
  - 示例：
  -    short short_num = 100;
- 22：signed
  - 声明有符号整形变量
  - 示例：
  -     signed int x = -10;
- 23：sizeof
  - 计算变量或类型占用的字节数
  - 示例：
  -     int size = sizeof(int);
- 24：static
  - 声明静态变量，其值在程序运行期间保持不变，且在函数外声明的变量具有文件作用域，在函数内声明的变量具有函数作用域
- 25：struct
  - 声明结构体类型，用于定义一组相关的变量
  - 示例：
    -     struct Student{
    -     char name[50];
    -     int age;};
- 26：switch 
  - 开始switch-case语句
  - 示例：
  -     switch(x){
  -     case 1://执行代码;break;
  -     case 2://执行代码;break;
  -     default://执行代码；
  -     }
- 27：typedef
  - 定义类型别名
  - 示例：
  -     typedef int Interger;
- 28：union
  - 声明共同体类型，用于定义一组变量共用一段内存
  - 示例：
  - 
- 29：unsigned 
  - 声明无符号整形变量
  - 示例:
  -     unsigned int x = 10;
- 30：void
  - 声明函数返回类型为空，或者声明函数参数为空
  - 示例：
  -     void func(){
  -     //执行代码;
  -     }
- 31：volatile
  - 声明易变变量，告诉编译器不要对其进行优化，以便在程序中对其直接进行读写操作
  - 示例：
  -     volatile int x = 10;
- 32：while
  - 开始while循环
  - 示例：
  -     while (condition){
  -     //循环体;
  -     }
### 标识符：
#### 作用：用户为各种自定义数据进行命名
#### 组成原则：
- (1):子母和数字以及下划线_组成
- (2):首字母只能时字母或者下划线_不能是数字
- (3):长度不能超过31个字符
- (4):不能使用32个保留字
#### 注意：系统是区分大小写的
## 8：算法简介：
### 算法定义：
- 广义上的算法：算法是解决问题的步骤，是对特定问题求解步骤的一种描述
- 计算机算法：使用程序解决问题的逻辑步骤，是指令的有限序列
#### **任何算法可以通过三种逻辑关系来进行描述**：
- 顺序结构
- 循环结构
- 分支结构
#### 正确的算法要满足的条件：
- 每一个逻辑步骤可以通过语句进行实现
- 每一个步骤之间的关系是唯一的
- 算法是要终止的，不能够产生死循环
##### 算法举例：
- q1:求解累加和大于阈值是最后一次所加的值
- q2：求解两个正整数的最大公约数
#### 算法的特性：
- 有穷性
- 确定性
- 有输入
- 有输出
- 可行性
- 高效性
- 健壮性
#### 算法的表示：
- 自然语言
- 伪代码法
- 流程图
- 计算机语言
## 9：程序设计的一般方法：
### 程序开发步骤
- 需求分析Analysis
- 设计Design
- 编码Write
- 编辑Design
- 编译Complie
- 连接Link
- 运行Run
- 调试Debug
### 开发C语言程序的步骤
- 设计算法
- 编写程序源代码
- 编辑源程序
- 编译和连接
- 调试程序，直到改正了所有的编译错误和运行错误
- 运行程序
### 数据：
- 静态数据：静态数据是指一些永久性的数据，一般存储在硬盘中。
- 动态数据：动态数据是指在程序运行过程中，动态产生的临时数据，一般存储在内存中。
- 存储的时长：计算机关闭之后，这些临时数据都会被清除
